# 🎬 Movie Data Analysis using Python

## 📌 Overview
This project analyzes a movie dataset to identify patterns in ratings and popularity. It is part of my learning journey in Data Analysis and Artificial Intelligence.

## 📊 Key Insights
- Most movie ratings fall between 3 and 4  
- Popular movies tend to receive a higher number of ratings  
- Highly rated movies are not always the most popular  

## 🛠 Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

## 📁 Dataset
MovieLens dataset

## 🚀 What I Learned

- Performing exploratory data analysis (EDA)  
- Visualizing data using graphs  
- Extracting meaningful insights from data
